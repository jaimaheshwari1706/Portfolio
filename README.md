## ğŸš€ Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Angular CLI

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd jai-portfolio
   ```

2. **Setup Frontend**
   ```bash
   cd frontend/portfolio
   npm install
   ```

### Running the Application

```bash
cd frontend/portfolio
ng serve
```
App runs on: http://localhost:4200
